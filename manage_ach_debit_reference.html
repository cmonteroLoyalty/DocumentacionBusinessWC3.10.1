Preferences

The widget has the following preferences that managers or administrators can edit in Experience Manager:

Label Name Value type Default value

Number of debits per page

pageSize

long

20

Max. number of navigation pages

maxNavPages

long

3

Number of seconds to dismiss the notification

notificationDismissTime

string

3

Select the method for loading more debit orders

loadMoreMethod

string

pagination

Schedule type for the debits

scheduleType

string

one-off

Preferences that have the Admin column checked, are only editable for administrators.
