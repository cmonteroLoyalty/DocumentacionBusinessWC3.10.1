Connect accounting software

Shared library

All Cash Flow Forecasting widgets use a shared library, see Cash Flow Forecasting common library.

Selector name

This widget has the following selector names:

  • No connection screen:

    bb-cash-flow-link-accounting-platform-initial

  • Set up connection modal:

    bb-cash-flow-select-accounting-platform

Required imports

The following imports are required to set up and customize this widget:

  • Widget module import

    Import the following modules into the module of your wrapper widget:

    import { CashFlowConnectAccountingSoftwareWidgetAngModule } from '@backbase/business-ang/cash-flow';

  • Data module imports

    Import the following data modules into the module of your wrapper widget:

    import { CashFlowDataHttpModule } from '@backbase/data-ang/cash-flow';

Extension slots

This widget has the following extension slots:

  • No connection screen:

    1. bbCashFlowLinkAccountingPlatformInitialCustomizable

  • Set up connection modal:

    1. bbCashFlowSelectAccountingPlatformCustomizable

    2. bbCashFlowSelectAccountingPlatformListCustomizable

    3. bbCashFlowSelectAccountingPlatformInfoCustomizable

    4. bbCashFlowSelectAccountingPlatformFooterCustomizable
