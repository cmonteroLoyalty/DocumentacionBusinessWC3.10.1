View FX pair details

Preferences

The widget has the following preferences that managers or administrators configure in Experience Manager:

Name

Description

Configurable for

Default value

autoUpdateInterval

Sets the interval to poll the server for new currency rates, in seconds.

Admin

15

output.backToPrevPage

The widget output once the customer clicks to return to the previous page.

Admin

navigation:

output.selectedCurrenciesPair

The widget output once the customer clicks Place order.

Admin

navigation:

typeOfCalculation

Sets the value to be used in the calculation for the candlestick chart. Set it to Ask/Bid, None, or Open/Close.

Admin

None

Extension slots

The extension slot for this widget is the following:

  1. bbFxRatesPairDetailWidgetCustomizable

  2. bbFxRatesFavoriteButtonCustomizable

  3. bbFxRatesAdaptiveCandleChartCustomizable

  bbFxRatesAdaptiveCandleChartCustomizable
  let-hostRef
