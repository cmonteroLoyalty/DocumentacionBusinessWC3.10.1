This page includes the following information:

  • Selector name

  • Required imports

  • Extension slots

  • API

  • Implementation example

Selector name

This widget has the following selector name: bb-messages-create-message-modal-widget-ang.

Required imports

The following imports are required to set up and customize this widget:

  • Angular Common module import:

    import { CommonModule } from '@angular/common';

  • Backbase Core module import:

    import { BackbaseCoreModule } from '@backbase/foundation-ang/core';

  • Widget module import:

    import { MessagesCreateMessageModalWidgetModule } from '@backbase/messages-create-message-modal-widget-ang';

Extension slots

This widget has the following extension slots:

  1. bbMessagesCreateMessageModalCustomizable

  2. bbCreateMessageFormCustomizable

  3. bbMessagesUploadAttachmentsCustomizable

bbMessagesCreateMessageModalCustomizable

This extension slot includes the following default template:

<ng-template

  let-host
  let-maxAttachmentSize="context.maxAttachmentSize"
  let-maxSubjectLength="context.maxSubjectLength"

      bbButton

      i18n="Button title for compose message@@messages.createMessage.compose"
      (click)="host.openModal()"
    >
      Compose
    </button>
  </div>

  <bb-modal-ui
    [isOpen]="host.modalOpened"
    (cancel)="host.closeModal()"
