Web widgets

Understand Products describes the functionalities of Products. The following web widgets deliver these functionalities:

  • List products

    • View business account list: View all accounts in table or card view

    • View favorite accounts: View favorite accounts in card view

  • Select products

    • Select business account: Select an account from a drop-down

  • View product details

    • View business account details: View the details of an account

    • View business account balance history: View the account balance history

  • Manage products

    • View favorite accounts: Add account as a favorite

View business account list

This widget shows the accounts of the bank customer in table or cards view.

By default, the widget shows all account types, but administrators can configure the widget in Experience Manager to filter a specific account type.

Card view:

Features

  • View overall aggregated balances of accounts, grouped by currency.

Select business account

The Select business account widget allows bank customer to select an account from their account list. The customer can select one, multiple, or all accounts. The drop-down shows the name and the current account balance.

If the customer selects multiple accounts the number of the selected accounts displays, along with their names. If the customer chooses to select all accounts, the view will show all accounts have been selected, and the total number of accounts in the selection.

Closed widget:

Open widget:

This widget is used in combination with several other widgets. When both widgets are placed on the same page, they can communicate with each other. For example, if the Select business account widget is placed on the same page as the View transaction list widget, and the bank customer selects a product, the Select business account widget publishes an event. The View transaction list widget subscribes to this event and changes its view to display the transactions of the selected product.

This widget interfaces with the Products DBS capability using the arrangement-client-api.

Features

The following features are configurable in Experience Manager:

  • Allow the bank customer to filter the arrangements that are presented, without additional server requests.

  • Allow the bank customer to select more than one arrangement.

View business account balance history

Bank customers use the View business account balance history widget to view the history of an account balance in graphical or table view.

This widget interfaces with the Products DBS capability using the arrangement-client-api.

Features

This widget provides the following features to the bank customer:

  • View the account balance history in table view.

  • View the account balance history in a graph.

Related topics

For further information, see:

  • Reference - View business account balance history

View business account details

Bank customers use the View business account details widget to view the details of an account. When they are finished, customers can return to the View business account list by selecting the back button.

This widget interfaces with the Products DBS capability using the arrangement-client-api.

Features

This widget provides the following features to the bank customer:

  • View account details.

  • Navigate back to accounts list.

Related topics

For further information, see:

  • Reference - View business account details

View favorite accounts

The View favorite accounts widget displays the favorite accounts of the bank customer in card view.

The horizontal list of favorite accounts is scrollable until the last favorite account is displayed.

To add an account to the favorites, the customer clicks the star icon associated with that account in the View business account list widget. It is possible to add accounts to the favorites from both list and card views.

Once an account is a favorite, the star icon associated with that account is filled. To remove an account from the favorites, the bank customer clicks the filled star icon either in the View favorite accounts View business account list widget.

If there are no favorite accounts, the widget is not visible.

This widget interfaces with the Products DBS capability using the arrangement-client-api.

Features

This widget has the following features:

  • Add accounts to the favorites.

  • Remove accounts from the favorites.

  • View the favorite accounts in card view.
