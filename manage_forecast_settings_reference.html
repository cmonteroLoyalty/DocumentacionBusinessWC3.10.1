Manage forecast settings

Selector name

This widget has the following selector name: bb-cash-flow-forecast-settings-list.

Required imports

The following imports are required to set up and customize this widget:

  • Widget module import

    Import the following modules into the module of your wrapper widget:

    import {
  CashFlowForecastSettingsInitialModule,
  CashFlowMainContainerModule,
  CashFlowBankingAccountListModule,
} from '@backbase/cash-flow-common';

Extension slots

This widget has the following extension slots:

  1. bbCashFlowManageForecastSettingsCustomizable

  2. bbCashFlowForecastSettingsListCustomizable

  3. bbCashFlowForecastSettingsCurrencyCustomizable

  4. bbCashFlowForecastSettingsBankingAccountsCustomizable

  5. bbCashFlowForecastSettingsAccountingCategoriesCustomizable

  6. bbCashFlowForecastSettingsAccountingCategoriesListCustomizable

  7. bbCashFlowBankingAccountListCustomizable

  8. bbCashFlowBankingAccountListItemCustomizable

    <ng-container cash-flow-widget-body>

        <div class="bb-block bb-block--md">

            heading="Forecasting settings"

            headingType="h2"

          </bb-header-ui>
        </div>
        <bb-cash-flow-forecast-settings-list
          *ngIf="forecastReferencesExists; else initialState"
        >
        </bb-cash-flow-forecast-settings-list>
      </div>
      <div class="bb-block bb-block--md">
        <bb-header-ui
          heading="Connected software"
          i18n-heading="
            Connected software header@@settings.connected-software.header"
          headingType="h2"
        >
        </bb-header-ui>
      </div>
      <div
        *ngIf="accountingPlatform"
        class="cash-flow-manage-forecast-settings-accounting-platform"
      >
        <img
          [src]="accountingPlatform.logoUrl"
          [alt]="accountingPlatform.name"
          data-role="cash-flow-manage-forecast-settings-accounting-platform__logo"
          class="cash-flow-manage-forecast-settings-accounting-platform__logo"
        />

    </bb-cash-flow-forecast-settings-initial>
  </ng-template>
</ng-template>

bbCashFlowForecastSettingsListCustomizable

This extension slot includes the following default template:

<ng-template
  bbCashFlowForecastSettingsListCustomizable
  let-hostRef
  let-forecastConfiguration="context.forecastConfiguration"
  let-accountingCategories="context.accountingCategories"
  let-bankingAccounts="context.bankingAccounts"
>
  <bb-cash-flow-forecast-settings-currency
    [currencyCode]="forecastConfiguration?.currencyCode"
    (changeCurrency)="hostRef.onChangeCurrency(forecastConfiguration)"
  >
  </bb-cash-flow-forecast-settings-currency>

  <bb-cash-flow-forecast-settings-banking-accounts
    [bankingAccounts]="bankingAccounts"
    (changeBankingAccounts)="
      hostRef.onChangeBankingAccounts(forecastConfiguration)
    "
  >
  </bb-cash-flow-forecast-settings-banking-accounts>

  <bb-cash-flow-forecast-settings-accounting-categories
    [accountingCategories]="accountingCategories"
    (changeAccountingCategories)="
      hostRef.onChangeAccountingCategories(forecastConfiguration)
    "
  >
  </bb-cash-flow-forecast-settings-accounting-categories>
</ng-template>

This extension slot consumes the following object properties from its custom context:

Object property Description

readonly bankingAccounts$: Observable<BankingAccount[]>

Stream of banking accounts.

readonly accountingCategories$: Observable<AccountingCategories>

Stream of accounting categories.

readonly forecastConfiguration$: Observable<ForecastConfiguration | undefined>

Stream of forecast configuration.

Method

Name Description

onChangeCurrency(configuration: ForecastConfiguration): void

Opens modal to configure forecast settings.

onChangeBankingAccounts(configuration: ForecastConfiguration): void

Opens modal to edit banking accounts.

onChangeAccountingCategories(configuration: ForecastConfiguration): void

Opens modal to edit accounting categories.
