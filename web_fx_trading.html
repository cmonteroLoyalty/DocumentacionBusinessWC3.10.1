Web widgets

The web widgets included in FX Trading are:

  • View FX rates

  • View FX pair details

  • Create FX trade order

View FX rates

In the View FX rates widget, the bank customer views currency exchange rates. They may search for currency pairs and clicks through to place a trade order.

The table includes the following information:

  • Currency: The currency code for the currency pair.

  • Last price: The last executed price for the currency pair.

  • Change: Today’s price change in green or red.

  • %Change: Today’s price change in percentage in green or red.

  • Bid: The bid price available for the currency pair within the system.

  • Ask: The ask price available for the currency pair within the system.

The state of the icon indicates whether the customer can trade the currency pair. This icon is clickable when the customer has at least one open account for each currency in the pair.

By clicking the currency symbol, the customer accesses the View FX pair details widget. By clicking anywhere else in the currency row, the customer can see detailed information about the pair. The provided information is the same as in the View FX pair details widget and configurable in Experience Manager.

The list is updated automatically using intervals specified in the widget preferences or when the customer refreshes the page or uses pagination.

Features

This widget provides the following features for the customer:

  • View currency exchange rates and their fluctuations.

  • Sort currencies in the list in the ascending or descending order.

  • Mark currency pairs as favorite and then see them in the Favorites tab.

  • View detailed information about a currency pair.

  • See whether the currency pair can be traded.

  • Search for currency pairs.

  • View the number of matching results.

  • Discard the search query.

  • Access the Create FX trade order widget by clicking the corresponding icon.

  • Access the View FX pair details widget by clicking a currency symbol.

The navigation between widgets is configurable in Experience Manager.

View FX pair details

In the View FX pair details widget, the bank customer views detailed information about a selected currency pair.

This information includes the following:

  • Currency symbol: The name of the currency pair according to international conventions.

  • Update time: The time when the exchange rate was last updated.

  • Price: The last executed price for the currency pair.

  • Change: Today’s price change in green or red, depending on the direction.

  • %Change: Today’s price change percentage in green or red, depending on the direction.

  • Candlestick chart: The fluctuations of the price.

  • Open: The initial price at the beginning of the current trading period.

  • Prev. Close: The final price at the end of the previous trading period.

  • YTD Return: The amount of profit or loss generated by trading the corresponding currency pair since the beginning of the current calendar year (usually January 1st), the first trading day of the year, or the first day of the fiscal year.

  • Bid: The bid price available for the currency pair within the system.

  • Ask: The ask price available for the currency pair within the system.

  • 52 Wk Range: The price range for the currency pair during a 52-week period.

  • Day’s Range: The price range for the currency pair starting from the beginning of the day.

The customer can access the Create FX trade order widget by clicking Place order or go back to the list by clicking Back to currencies.

The information is automatically refreshed using the intervals set up in the widget preferences.

Features

This widget provides the following features for the customer:

  • View detailed information about a selected currency pair.

  • Access the Create FX trade order widget by clicking Place order.

  • Access the View FX rates widget by clicking Back to currencies.

The navigation between widgets is configurable in Experience Manager.

Create FX trade order

In the Create FX trade order widget, the bank customer creates, reviews, and submits a trade order for a selected currency pair.

The widget includes the following information:

  • Currency symbol: The name of the currency pair according to international conventions.

  • Update time: The time when the exchange rate was last updated.

  • Bid: The bid price available for the currency pair within the system.

  • Ask: the ask price available for the currency pair within the system.

  • Selling amount: The amount to be sold.

  • Buying amount: The amount to be bought.

  • Paying from: The bank account to be used for buying the currency.

  • Paying to: The bank account to deposit funds after selling the currency.

The rate is updated when the customer refreshes the page or clicks Next to review order details. In both cases, the buying and selling amounts specified by the customer are recalculated automatically.

If the rates change when the customer clicks Next to review the order, the widget displays a warning message with the new rates, asking the customer to confirm or edit the order.

The review screen includes the following information:

  • Current price: The current price for the currency pair.

  • Availability time: The time that the price remains available with a progress bar.

  • Date and Time: The time and date of creating the order.

  • Selling: The amount of currency to be sold.

  • Buying: The amount of currency to be bought.

  • Paying from: The account to be used for buying the currency.

  • Paying to: The account to deposit the funds after selling the currency.

The customer can submit the order, discard it, or return to edit it. If the time allocated to submit the order expires, the widget displays a warning message and offers the customer to go back to editing.

Features

This widget provides the following features for the bank customer:

  • Review the exchange rate for the currency pair.

  • Specify the selling and buying amounts.

  • Swap the base and quote currency.

  • Specify the accounts to withdraw and deposit funds.

  • Review updated exchange rates if they have changed in the process of creating the order.

  • Have the buying and selling amounts recalculated automatically according to the updated rates.

  • See a warning if the selected Paying from account does not have sufficient funds for the transaction.

  • See warnings if any of the required fields have not been filled out.

  • Cancel the creation of a trade order.

  • Review the details of the trade order.

  • Submit the trade order.
