Web widgets

Understand Contacts describes the functionalities of Contacts. The following web widgets deliver these functionalities:

Manage contacts

  • Create business contact

  • Edit business contact

  • Manage contact details

  • View contact list

  • Manage business contacts

Approval flow for changes to contacts

  • Manage contact change requests

  • Approve contact change requests

Create business contact

The Create business contact allows the current user to create a contact.

Create business contact interfaces with the Contacts DBS capability using the contact-manager-client-api.

Features

This widget has the following features:

  • Create a contact.

Related topics

For further information, see:

  • Reference - Create business contact

Manage contact details

The Manage contact details allows the current user to view and/or delete contact information.

Manage contact details interfaces with the Contacts DBS capability using the contact-manager-client-api.

Features

This widget has the following features:

  • View contact details.

  • Delete a contact.

Related topics

For further information, see:

  • Reference - Manage contact details

View contact list

The View contact list allows a user to view all the contacts stored in the system. Contacts are grouped by the first letter of the name and in alphabetical order. Search can be executed on the list to find contacts by certain term.

View contact list interfaces with the Contacts DBS capability using the contact-manager-client-api.

Features

This widget has the following features:

  • View the contact list.

  • Search contacts.

Related topics

For further information, see:

  • Reference - View contact list

This widget interfaces with the Contacts DBS capability using the contact-manager-client-api.

Features

  • Edit a contact.

Related topics

For further information, see:

  • Reference - Edit business contact

Manage business contacts

In the Manage business contacts, the user manages a list of frequently used counter-parties, for example, payees.

Manage business contacts displays previously saved contacts. The user is able to access these contacts depending on their permissions.

For each contact the name and the IBAN number is shown. If the user has the right permissions, they are able to edit, delete and search contacts.

IBAN validation ensures that only correctly formatted IBANs are accepted, although confirmation of their existence requires connecting to the core banking system.

Manage business contacts interfaces with the Contacts DBS capability using the contact-manager-client-api.

Features

This widget has the following features:

  • View a list of contacts and associated details.

  • Delete a contact.

  • Edit a contact.

  • Load contacts incrementally.

  • Search for a contact.

  • Distinguish incorrectly filled or empty mandatory fields highlighted by the widget.

Related topics

For further information, see:

  • Reference - Manage business contacts

Manage contact change requests

In Manage contact change requests, the current user views their own submitted change requests for contacts that are pending for approval.

To use Manage contact change requests, Approvals must be enabled for Contacts. If Approvals is enabled, change requests such as creating, editing, and deleting a contact require single or multi-approvals depending on the assigned approval policy.

Once a request is submitted for approval, the user is able to view it under My Requests. The list includes the following details for each item:

  • Status - contact status, Entered, Rejected, or Partially Authorized.

  • Name - contact name.

  • Request - description of the request, that is, the latest change that was initiated for the contact.

  • Creation Date - the timestamp of the latest change that was initiated for the contact.

The list items are sorted by Name alphabetically.

To view the contact details for a particular request, the user clicks the request in the list.

Details include the following:

  • Approval log - specifies the approval status, the name of the approver, and timestamp of the approval if applicable. It is also indicated if further approval is required. This section of the modal dialog is collapsed by default.

To cancel a pending contact request, and to delete a rejected contact request, the user clicks the icon associated with the request.

Manage contact change requests interfaces with the Contacts DBS capability using the contact-manager-client-api.

Features

This widget has the following features:

  • View the pending contact requests submitted by the current user.

  • Cancel a pending contact request.

  • Delete a rejected contact request.

  • View the contact details for an item in the list.

  • View the approval log for a contact request in details view.

Related topics

For further information, see:

  • Reference - Manage contact change requests

Approve contact change requests

In Approve contact change requests, the current user views a list of submitted updates to the contact list that are awaiting authorization.

Approve contact change requests interfaces with the Contacts DBS capability using the contact-manager-client-api.

If the current user has sufficient access control permissions, they click on a list item to review the details of the contact request.

The details include the following:

  • Approval log - specifies the approval status, the name of the approver, and the timestamp of the approval if applicable. It is also indicated if further approval is required. This section of the modal dialog is collapsed by default.

To approve or reject the contact request in details view, the user clicks Approve or Reject respectively.

To approve or reject the contact request in list view, the user clicks ✔ or ✗ next to the item respectively.

Features

This widget has the following features:

  • View a list of contacts request pending to approve

  • View the contact request details

  • View the approval log for a pending contact request in details view

  • Approve the contact request

  • Reject the contact request

Related topics

For further information, see:
