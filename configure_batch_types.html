Add a custom batch type to the batch payment widgets

Backbase provides example implementations for the following batch types out of the box: SEPA credit CSV file, NACHA ACH credit fixed-length file, and NACHA balanced ACH debit fixed-length file. To support a custom batch type, finish the following tasks:

  1. Add the custom batch type to the backend service

    Use either of the following methods:

    • Configuration: No coding needed, just create a YAML file that defines the parser and validator. No service restart needed. Parsed batch files are stored and exposed to integration services.

    • Java method: More freedom to create a parser and validator and place them on the classpath of the service. Restart of the service is necessary. Parsed batch files are not stored out of the box.

  2. Add a custom batch type to the batch payment widgets (this tutorial)

This tutorial shows you how to customize the widgets to handle custom batch types and includes the following sections:

  • Understand the configuration

  • Prerequisites

  • Add a custom batch type to the Upload batch payment widget

  • Add a custom batch type to the Log batch payment widget

  • Add a custom batch type to the Batch Manager widget

Understand the configuration

The default value of the BB_BATCH_TYPES configuration provider in Upload batch payment is:

export const BatchTypes: Array<BatchType> = [
  {
    type: 'BB_SEPACT_CSV',
    sizeLimit: '100',
    format: '.csv',
    iconClasses: 'bb-sepa',
  },
  {
    type: 'NACHA_CREDIT',
    sizeLimit: '50',
    format: '.ach',
    iconClasses: 'ach',
  },
  {
    type: 'NACHA_DEBIT',
    sizeLimit: '50',
    format: '.ach',
    iconClasses: 'ach',
  },
];

The interface for BatchType is available if you use the following include:

import { BatchType } from '@backbase/business-ang/batch';

The items in the interface are:

  • type: Mandatory. Use the type configured on the backend; in DBS under the backbase.payments.batches.supported-batch-upload-types setting.

  • sizeLimit: Mandatory. Maximum file size in Mb.

  • format: Mandatory. Expected file extension.

  • iconClasses: Optional. Name of styling class to show the icon.

Prerequisites

  • A project set up in a local or shared developer environment that contains the Upload Batch widget.

  • A custom batch type on the backend, see either of the following methods:

    • Configuration

    • Java method

Add a custom batch type to the Upload batch payment widget

  1. To customize Upload batch payment to support custom batch types, override the value of the BB_BATCH_TYPES config provider.

    To override the default value:

    1. Create a new file config.providers.ts.

    2. Configure the publicly exposed token:

      import { Provider } from '@angular/core';
import { BB_BATCH_TYPES } from '@backbase/business-ang/batch';

export const BatchTypeProvider: Provider = {
  provide: BB_BATCH_TYPES,
  useValue: [
    {
      type: 'BB_SEPACT_CSV',
      sizeLimit: '100',
      format: '.csv',
      iconClasses: 'bb-sepa',
    },
    ...
    {
      type: 'CUSTOM_BATCH_TYPE',
      sizeLimit: '100',
      format: '.json',
      iconClasses: 'custom-batch-icon-class',
    },
  ],
};

    3. Include the configuration in your Angular app. Import BatchTypeProvider from config.providers.ts and add it as a provider in app.module.ts:

      ...
import { BatchTypeProvider } from './config.providers';
...

@NgModule({
  ...
  providers: [BatchTypeProvider],
  ...
})
export class AppModule {}

  2. To display the custom batch type in the widget, customize the extension slot bbBatchTypeCustomizable. For example:

    <ng-template bbBatchTypeCustomizable let-hostRef let-batch="context.batch" let-last="context.last">
  <div class="bb-inline-stack bb-inline-stack--density-sm">
    <bb-icon-ui
      class="bb-inline-stack__item"
      size="md"
      name="file-{{ batch.type === 'BB_SEPACT_CSV' ? 'bb-sepa' : format.substring(1) }}"
      *ngFor="let format of batch.format.split(',')"
    ></bb-icon-ui>
    <span class="bb-inline-stack__item bb-text-bold" data-role="batch-types" i18n="@@bb-select-batch-file-type">
      {batch.type, select, BB_SEPACT_CSV {Backbase SEPA CT} NACHA_CREDIT {ACH Credit (Balanced)} NACHA_DEBIT {ACH Debit (Balanced)} other {Unknown batch type}}
    </span>
  </div>
  <div
    class="bb-text-support"
    i18n="Batch type description|Description for batch type@@select-batch-files.batch-type.description"
  >
    Up to {{ batch.sizeLimit }} Mb files in *{{ batch.format }} format.
  </div>
</ng-template>
