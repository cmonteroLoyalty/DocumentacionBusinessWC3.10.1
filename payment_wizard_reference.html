Initiate payment wizard

The payment wizard includes uses existing payment configs from the omni payment widget so all the customisations and configurations that are applicable to omni payment widget are supported the payment wizard widget.

Preferences

As the Payment wizard widget is a headless widget it doesn’t support widget preferences out-of-the-box, but the wizard configuration options can be added as preferences in the wrapper widget and can be configured by managers or administrators in Experience Manager

Extension slots

This widget doesn’t support extension slots.
