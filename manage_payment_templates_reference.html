Selector name

This widget has the following selector name:

bb-payord-manage-payment-templates-widget

Required imports

The following imports are required to set up and customize this widget:

  • Widget module import

    Import the following modules into the module of your wrapper widget:

    import {
  IconModule,
  SearchBoxModule,
  DropdownSingleSelectModule,
  EmptyStateModule,
  PaginatorModule,
  LoadingIndicatorModule,
  AmountModule,
} from '@backbase/ui-ang'

Access Control

A permission is the combination of a business function and a single privilege granted to a user on a given resource type. Privileges are enforced by Digital Banking Services. Widget behavior is affected by the permissions of the user accessing the widget.

The following table describes how the widget behavior changes depending on the permissions of the user:

Resource

Business function

Privilege

Widget behavior

Payments

Payment Templates

view

The current user views a list of saved payment templates.

If the user does not have the permission to view, the widget shows an empty list.

delete

The current user is able to see the button to delete saved payment template from the list.

If the user does not have the permission to delete, the delete button will be unavailable.
