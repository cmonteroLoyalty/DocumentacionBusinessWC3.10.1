Selector name

This widget has the following selector name: bb-product-summary-account-details-widget

Required imports

The following imports are required to set up and customize this widget:

  • Widget module import

    Import the following modules into the module of your wrapper widget:

    import { ProductSummaryAccountDetailsWidgetModule } from '@backbase/product-summary-account-details-widget-ang';
