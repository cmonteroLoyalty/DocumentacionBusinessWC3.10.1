Access Control

A permission is the combination of a business function and a single privilege granted to a user on a given resource type. Privileges are enforced by Digital Banking Services. Widget behavior is affected by the permissions of the user accessing the widget.

The following table describes how the widget behavior changes depending on the permissions of the user:

Resource

Business function

Privilege

Widget behavior

Batch

Batch - SEPA CT

create

The current user is able to upload a SEPA credit CSV file. If the user does not have the permission to create, the upload will fail.

Batch - ACH Credit

create

The current user is able to upload a NACHA ACH credit fixed-length file. If the user does not have the permission to create, the upload will fail.

Batch - ACH Debit

create

The current user is able to upload a NACHA balanced ACH debit fixed-length file. If the user does not have the permission to create, the upload will fail.

This widget has the following extension slots:

  2. bbInitiateBatchUploadCustomizable

bbInitiateBatchUploadContainerCustomizable

This extension slot includes the following default template:

<ng-template bbInitiateBatchUploadContainerCustomizable let-hostRef>
  <bb-initiate-batch-upload
    (filesSubmit)="hostRef.submitFiles($event)"
    [batchTypes]="hostRef.batchTypes"
    [initUpload]="hostRef.initUpload"
    (modalDismiss)="hostRef.onModalDismiss()"
  >
  </bb-initiate-batch-upload>

  <bb-duplicate-batch-upload-modal
    (submitDuplicateBatch)="hostRef.onSubmitDuplicateBatch($event)"
    (deleteDuplicateBatch)="hostRef.onDeleteDuplicateBatch($event)"
    [duplicateBatchDetails$]="hostRef.duplicateUpload$"
    [isModalOpen]="hostRef.isDuplicateUpload"
    (modalDismiss)="hostRef.dismissDuplicateUploadModal()"
  >
  </bb-duplicate-batch-upload-modal>
</ng-template>

bbInitiateBatchUploadCustomizable

This extension slot includes the following default template:

<ng-template bbInitiateBatchUploadCustomizable let-hostRef>
  <bb-modal-ui
    [isOpen]="hostRef.initUpload"
    [modalOptions]="hostRef.modalOptions"
    (dismissCallback)="hostRef.closeModal()"
  >
    <bb-modal-body-ui>
      <ng-template bbCustomModalBody>
        <ng-template [ngIf]="!hostRef.files" [ngIfElse]="confirm_Submit">
          <bb-select-batch-type
            [batchTypes]="hostRef.batchTypes"
            (modalDismiss)="hostRef.closeModal()"
            (fileSelect)="hostRef.selectFiles($event)"
          >
          </bb-select-batch-type>
        </ng-template>

        <ng-template #confirm_Submit>
          <bb-confirm-batch-submit
            (modalDismiss)="hostRef.closeModal()"
            (fileSubmit)="hostRef.submitFiles()"
            [selectedFiles]="hostRef.files"
            (fileUnselect)="hostRef.selectFiles($event)"
            role="alert"
          >
          </bb-confirm-batch-submit>
        </ng-template>
      </ng-template>
    </bb-modal-body-ui>
  </bb-modal-ui>
</ng-template>
