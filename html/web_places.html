Web widgets

The web widget included in Places is:

  • Find branches and ATMs - view the locations of brick-and-mortar branches or ATMs on a map.

Find branches and ATMs

This widget shows the bank customer the locations of brick-and-mortar branches, ATMs, or any other locations related to banks on a map and in a list.

The widget attempts to establish the bank customerâ€™s location. If the customer’s location is successfully established, the widget shows branches and ATMs that are close to the customer on a map and in a list.

The bank customer can specify a location other than their current one by using the search facility, or by dragging inside the map. They can change the initial map zoom level with the zoom controls. The current view radius of the map is based on its zoom level.

If the widget cannot establish the bank customer’s location, it uses default coordinates. Administrators can configure these coordinates in Experience Manager, as well as the distance within which the widget displays branch or ATM locations.

If the widget displays both list and map views, the list shows only the locations that are currently visible on the map. Frontend engineers can remove the map view through an extension slot. If there is no map view, the list only shows locations that are within a given radius from the location of the current user. The list is limited to a certain amount of items. Administrators configure the radius and the list limit in Experience Manager.

Features

  • View the nearest locations in a map view.

  • View detailed information about the location.

  • View opening times of the selected location.

  • Get directions to the selected location.

  • View the nearest locations in a list view.

  • Search for a location.

  • Search for an address in list view.

  • View places in a combined view, that is, on a map and as a list.

  • Filter places by location type (branch, ATM, etc.).
