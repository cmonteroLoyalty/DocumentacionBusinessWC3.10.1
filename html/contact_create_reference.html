Label Name Value type Default value

Milliseconds to dismiss any notification message

notificationDismissTime

long

5000

Set the access context scope

contactAccessContextScope

string

USER

Access Control

A permission is the combination of a business function and a single privilege granted to a user on a given resource type. Privileges are enforced by Digital Banking Services. Widget behavior is affected by the permissions of the user accessing the widget.

The following table describes how the widget behavior changes depending on the permissions of the user:

Resource

Business function

Privilege

Widget behavior

Contacts

Contacts

create

The current user is able to create a contact

If the user does not have the permission to create, an error occurs when a submission is made.

Extension slots

This widget has the following extension slots:

  1. bbCreateContactContainerCustomizable

  2. bbCreateContactFormCustomizable

bbCreateContactContainerCustomizable

This extension slot includes the following default template:

<ng-template bbCreateContactContainerCustomizable let-hostRef>
  <bb-modal-ui
    [modalOptions]="hostRef.modalOptions"
    [isOpen]="hostRef.isCreateContact">
    <bb-modal-body-ui>
      <ng-template bbCustomModalBody>
        <bb-create-contact-form
          [notificationDismissTime]="hostRef.notificationDismissTime"
          [accessContextScope]="hostRef.accessContextScope"
          [resetForm]="hostRef.resetForm"
          (createContactSuccess)="hostRef.onCreateContactSuccess()"
          (exitCreateContact)="hostRef.onCancelCreateContact($event)">
        </bb-create-contact-form>
      </ng-template>
    </bb-modal-body-ui>
  </bb-modal-ui>

  <bb-confirm-action-dialog
    [isOpen]="hostRef.isCancelCreateConfirmOpened"
    [confirmButtonColor]="'danger'"
    (confirm)="hostRef.onExitCreateContact()"
    (cancel)="hostRef.onResumeCreateContact()"
  >
    <span title i18n="@@contact-create.confirm-action-dialog.title">Discard entered data?</span>
    <span body i18n="@@contact-create.confirm-action-dialog.body">Are you sure to exit creation of a new contact? The data you have entered will be lost.</span>
    <span cancelButtonLabel i18n="@@contact-create.confirm-action-dialog.button.cancel">Not now</span>
    <span confirmButtonLabel i18n="@@contact-create.confirm-action-dialog.button.confirm">Discard</span>
  </bb-confirm-action-dialog>
</ng-template>
