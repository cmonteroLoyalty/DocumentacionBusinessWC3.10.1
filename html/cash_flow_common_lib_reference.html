Cash Flow Forecasting common library

Base components

Base components provide a common inheritance scheme, which purpose is to isolate logic and avoid duplication of the code with exactly the same user flow for different widgets.

LinkStatusBaseComponent

An abstract class isolate logic with checking and setting status of the linking with the accounting software through the connection to own backend service. This is part of the initial flow for all of the cash flow widgets, to understand inheritance clearly check the inheritance scheme.

import { LinkStatusBaseComponent } from '@backbase/business-ang/cash-flow';

Extends DestroyableComponent.

Properties

Name Description

smeLinkStatus$: Observable<LinkStatus>

Stream of SME link statuses

Methods

Name Description

protected setSmeStatusAsAuthorized(): void

Updates the link status to be Authorized

protected abstract checkLinkStatus(status: LinkStatus): void

Method is used to check status on every SME link status update

Models

  • CallState represents the status of API call. Possible values:

    • INIT

    • LOADING

    • LOADED

    • error: Error message

  • FetchingCallState represents the status of API call. Possible values:

    • INIT

    • LOADING

    • LOADED

    • FETCHING

    • error: Error message
