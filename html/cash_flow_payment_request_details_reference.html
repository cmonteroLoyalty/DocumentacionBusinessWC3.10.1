Cash flow payment request details

Selector name

This widget has the following selector name: bb-cash-flow-payment-request-details-modal.

Required imports

The following imports are required to set up and customize this widget:

  • Widget module import

    Import the following modules into the module of your wrapper widget:

    import {
  AmountModule,
  BadgeModule,
  BbDatePipeModule,
  ButtonModule,
  EmptyStateModule,
  IconModule,
  LoadingIndicatorModule,
  ModalModule,
} from '@backbase/ui-ang';

Extension slots

This widget has the following extension slots:

  1. bbCashFlowDetailsCustomizable

  2. bbCashFlowItemDetailsModalCustomizable

bbCashFlowDetailsCustomizable

This extension slot includes the following default template:

<ng-template
  bbCashFlowPaymentRequestDetailsCustomizable
  let-paymentItemId="context.paymentItemId"
  let-itemView="context.itemView"
  let-hostRef
>
  <bb-cash-flow-payment-request-details-modal
    *ngIf="paymentItemId && itemView"
    [paymentItemId]="paymentItemId"
    [itemView]="itemView"
    (closeModal)="hostRef.onCloseModal()"
  >
  </bb-cash-flow-payment-request-details-modal>
</ng-template>

This extension slot consumes the following object properties from its custom context:

Object property Description

paymentItemId: Observable<string | undefined>

Stream of selected payment request item ID.

itemView: Observable<CashFlowItemView | undefined>

Cash flow item view.

Object property Description

isModalOpen: boolean

Stream of the flag that details modal is open.

Stream of custom error.

paymentItemDetails: Observable<CashFlowItemDetailsData>

Stream of payment item details request.

isLoading: boolean

Stream indicates that payment details request is pending.

Preferences

Title and description Name Configurable for Type Default value

input.paymentItemId

Parameter name that passes the payment ID from the Manage payables and the View receivables widgets to this widget, the View payment request details. The string you enter here should match the output parameter with the same name in the Manage payables and the View receivables widgets.
