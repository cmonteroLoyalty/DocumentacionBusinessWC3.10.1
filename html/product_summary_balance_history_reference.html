Selector name

Required imports

The following imports are required to set up and customize this widget:

Import the following modules into the module of your wrapper widget:

import { LineChartModule } from '@swimlane/ngx-charts';
import { ReactiveFormsModule } from '@angular/forms';
import { ProductSummaryBalanceHistoryWidgetModule } from '@backbase/product-summary-balance-history-widget-ang';

This extension slot consumes the following object properties from its custom context:

Object property Description

balanceHistorySeries: BalanceHistoryDataSeries[]

Balance history chart area data.

balanceHistoryPaginated: BalanceHistoryPaginated

Balance history table data.

paginationParams: BalanceHistoryPaginationParams

Pagination data for chart and table.

isSmallScreen: boolean

Specifies whether screen size is small. (true if element’s width smaller or equal to 640px).

loadingSeries: boolean

Defines if chart data fetching is in progress.

loadingPaginated: boolean

Defines if table data fetching is in progress.

accountId: string

Selected account id. Obtained from the routing parameters.

requestParamsSeries: GetBalanceHistoryRequestSeriesObjectType

Request parameters for getting balance history chart items.

errorSeries: BalanceHistoryError

Contains chart data fetching errors.

errorPaginated: BalanceHistoryError

Contains table data fetching errors.

showChartLegend: boolean

Specifies whether chart legend should be displayed or not. Obtained from input showChartLegend.

multiple

Sets whether the bank customer can see details for multiple accounts at a time.
