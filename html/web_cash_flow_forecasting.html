Web widgets

Note
 This section describes an add-on. You may use this add-on only if you have purchased an appropriate license for this add-on based on a signed Product Order Form.

The web widgets included in this capability are:

  • Connect accounting software: Displays the flow of connecting to accounting software for Cash Flow Forecasting.

  • Configure forecast: Configure a forecast wizard.

  • Manage payables: Displays a list of available payables of a company, based on the information retrieved from the accounting software.

  • Cash flow payment request details: Displays modals which detail specific information about a selected payable or recievable.

  • View cash flow forecast: Display the forecast bar chart.

  • View receivables: Displays a list of available receivables of a company, based on the information retrieved from the accounting software.

  • Manage forecast settings: Displays the configuations and settings for Cash Flow Forecasting, and for the linked accounting software.

Connect accounting software

Connect accounting software displays the flow of connecting the user to their accounting software for Cash Flow Forecasting.

Cash Flow Forecasting uses Codat, a third-party software to aggregate the financial data from the online accounting systems.

The user connects to their accounting software through Connect accounting software. If there is no active connection, the widget displays a button to open up a modal that guides the user trough setting up the connection.

The financial data must be supplied by a linked provider of third-party software.

The user must select and log in to the supported accounting software in order to create a dedicated SME company with Codat. If there are multiple users with access to the accounting software and Cash Flow Forecasting, they will be linked to this SME company.

If the user’s log in is successful, the data fetching from the accounting software will be initiated.

Configure forecast

The Configure forecast widget allows the user to configure Cash Flow Forecasting.

Features

  • Select forecast currency

    The forecast currency determines which accounts should be included in the forecast. The list of available currencies is taken from the list of open accounts.

    US Dollar is preselected by default. If US Dollar is not available, then the first currency is taken from the list of available currencies.

    Multi-currency or currency conversion is not supported, meaning it is not possible to create a forecast that includes accounts in multiple currencies.

  • Select accounts: Select one or multiple accounts to have their balances included in the forecast.

  • Filter the list of accounts by the account name.

  • Select accounting categories that you want to include in the forecast.

  • Filter the list of accounting categories by the account name.

Manage payables

The Manage payables widget allows the user to view a list of their company’s available payables.

When the user views the list of payables, the widget displays all payables available to the customer in the accounting software. The user will also view the timestamp of the last time the interface synced with the accounting software, showing that the information is correct as of the time and date displayed. Each listed item, a receivable, includes:

  • Status: The current state of the bill.

  • Issue date: The issue date of the bill.

  • Due date: The date when the bill must be fully paid.

  • Currency: The currency of the bill.

  • Amount due: The amount of funds the business expects to pay to a supplier.

  • Total amount: The amount of the bill, inclusive of tax.

  • Supplier: The name of an entity that issued the bill.

  • Category: The account that the business uses to record transactions.

Payables with only a portion of the full amount paid have the status Partiallypaid. In the Amount due, the user sees the outstanding amount to be paid out of the total amount of the invoice.

The user may filter the list of payables by status, such as:

  • All: Displays the entire list, set as default.

  • Open

  • Paid

The user may isolate listed payables further by initiating a search of a supplier name or reference number.

Features

  • View list of payables.

  • View the due amount.

  • View the time of last update.

  • Filter list of payables.

  • Search list of payables.

Cash flow payment request details

The Cash flow payment request details widget displays modals which detail specific information about a selected payable or recievable. The widget retrieves the required infomation from the user’s connected accounting software via Codat.

The modal window displays each receivable or payable with the following list items:

  • ID: A unique identifier of an invoice in the accounting system.

  • Status: The current state of the invoice

  • Issue date: The date when an invoice was issued.

  • Due date: date when an invoice has to be fully paid.

  • Expected date: The date when an invoice is expected to be paid.

  • Currency: The currency of the invoice.

  • Amount due: The amount of funds the business expects to receive or pay.

  • Paid amount: The amount of funds the business expects to receive or pay.

  • Total amount: The amount of the invoice, inclusive of tax.

  • Customer name: The name of an entity that an invoice was issued to.

  • Category: The account a business uses to record accounting transactions.

View cash flow forecast

The View cash flow forecast widget shows the forecast for the cash flow in the form of a bar chart and a table.

The widget requires:

  • Active connection to an accounting provider: Connect accounting software

  • Forecast configured: Configure forecast

  • Accounting data: If there is an active connection to an accounting provider, but that does not return any accounting data to Cash Flow Forecasting, the widget shows a message that explains what is missing.

Features

  • View historic and future closing balances plotted on a bar chart

  • View balance and cash flow details in a table

  • Selector for the period:

    • Daily (default): Shows the last 7 days and the upcoming 14 days

    • Weekly: Shows the last 4 weeks and the upcoming 10 weeks

    • Monthly: Shows the last 3 months and the upcoming 12 months

Bar chart details

The bar chart has the following specific features:

  • X-axis

    • The x-axis is always visible.

    • The x-axis is scrollable.

    • The increments on the x-axis are daily, weekly, or monthly and correspond to the selected period.

    • The labels of the increments on the x-axis are as follows:

      • Daily: Month day, year. For example:

        • Nov 21, 2017

        • Jan 5, 2020

      • Weekly: W number of the week, year. For example:

        • W3, 2017

        • W54, 2018

      • Monthly: Month year. For example:

        • Jan 2018

        • Feb 2018

  • Y-axis

    • The y-axis is never visible. The balance values are displayed above each bar.

    • The increments of the y-axis scale with the the balance values in the current bar chart.

  • Grid lines

    • The horizontal grid lines are always visible and align with the y-axis increments.

    • Vertical grid lines are never visible.

  • Tooltip

    • Trigger the tooltip with a mouse hover (desktop) over or a tap (mobile) on the chart. This action shows the tooltip with information on the selected bar and highlights the selected bar.

    • The tooltip shows different information depending on the timeframe of the selected bar:

      Past balance

      Jan 1, 2019

      Balance: USD 1,234.00

      Current balance

      Feb 4, 2020

      Current: USD 10,123.00

      Forecast: USD 11,124.00

      Future balance

      Mar 1, 2020

      Forecast: USD 900,231.00

    • Dismiss the tooltip by navigating the mouse away or by a second tap.

Table details

The table below the bar chart displays the following information:

  • Opening Balance: The balance at the beginning of the period increment.

  • Cash In: Total amount of receivables in this period increment. Expand to split it out into category groups and categories.

  • Cash Out: Total amount of payables in this period increment. Expand to split it out into category groups and categories.

  • Net Change: The net amount of receivables minus payables.

  • Closing Balance: The balance at the end of the period increment.

Manage forecast settings

The Manage forecast settings widget displays the configuations and settings for Cash Flow Forecasting and for the linked accounting software.

By selecting the Settings tab, the user is able to view and edit the settings of a selected forecast. The flow for editing a forecast follows the same path as creating a forecast, requiring a currency, banking accounts, and accounting categories or accounts.

Features

  • View forecast settings.

    The widget displays the forecast settings the user provided when creating the a forecast. The settings are:

    • Currency of the selected forecast.

    • The list of bank accounts included in the selected forecast.

    • The list of accounting categories and accounts included in the selected forecast.

  • Edit forecast settings.

    The user can use Manage forecast settings to edit a selected forecast.
