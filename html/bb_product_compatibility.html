The products marked with semver use semantic versioning. Other Backbase products use the following versioning structure:

<product_line>.<major_version>.<minor_version>.<patch_version>

All minor versions of Backbase products within their related major version are compatible. For example, Retail Banking Widget Collection 3.4.x is compatible with DBS 2.16.x and CXS 6.1.x.

The table above lists the minor versions that have been verified for a certain release. As a best practice, new projects should use the latest minor version of all products and modules within the most recent compatible major version. This best practice also applies to projects that are upgrading.
