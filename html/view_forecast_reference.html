View cash flow forecast

Shared library

All Cash Flow Forecasting widgets use a shared library, see Cash Flow Forecasting common library.

Selector name

This widget has the following selector name: bb-cash-flow-forecast-widget-ang.

Required imports

The following imports are required to set up and customize this widget:

  • Widget module import

    Import the following modules into the module of your wrapper widget:

    import { CashFlowForecastWidgetAngModule } from '@backbase/business-ang/cash-flow';

  • Data module imports

    Import the following data modules into the module of your wrapper widget:

    import { CashFlowDataHttpModule } from '@backbase/data-ang/cash-flow';

Extension slots

This widget has the following extension slots:

  1. bbCashFlowForecastCustomizable

  2. bbCashFlowForecastTableCustomizable

  3. bbCashFlowForecastCadenceSelectorCustomizable

  4. bbCashFlowForecastBarChartCustomizable

  5. bbCashFlowForecastChartBarCustomizable

  6. bbCashFlowForecastTooltipBodyCustomizable

bbCashFlowForecastCustomizable

This extension slot includes the following default template:

<ng-template
  bbCashFlowForecastCustomizable
  let-hostRef
  let-isError="context.isError"
  let-isLoading="context.isLoading"
  let-forecastReferencesExists="context.forecastReferencesExists"
  let-forecastData="context.forecastData"
  let-syncDatetime="context.syncDatetime"
>
  <bb-cash-flow-main-container
    [isErrorState]="isError"
    [isLoadingState]="isLoading"
    [isEmptyState]="forecastData && !forecastData?.length"
    [contentWrapperClasses]="null"
  >
    <ng-container cash-flow-widget-body>
      <ng-container
        *ngIf="forecastReferencesExists && forecastData; else initialState"
      >
        <div class="bb-cash-flow-forecast">
          <bb-cash-flow-forecast-table
            [forecastData]="forecastData"
          ></bb-cash-flow-forecast-table>
          <bb-cash-flow-last-updated-time
            [dateTimestamp]="syncDatetime"
            wrapperClasses="bb-cash-flow-forecast__creation-date"
          >
          </bb-cash-flow-last-updated-time>
        </div>
        <div
          class="bb-state-container bb-cash-flow-forecast-mobile-state"
          data-role="cash-flow-forecast-mobile-view-empty-state"
        >
          <bb-empty-state-ui
            title="Available on a higher resolution"
            i18n-title="
              Title for mobile view empty
              state@@forecast.state.mobileView.emptyState.title"
            subtitle="You can view this page on a screen with resolution higher than 576px."
            i18n-subtitle="
              Message for mobile view empty
              state@@forecast.state.mobileView.emptyState.message"
            iconModifier="computer"
            iconSize="xxl"
          ></bb-empty-state-ui>
        </div>
      </ng-container>
    </ng-container>
  </bb-cash-flow-main-container>

  <ng-template #initialState>
    <bb-cash-flow-forecast-settings-initial
      containerClasses="bb-state-container--min-height"
      (setupForecast)="hostRef.onSetupForecast()"
    >
    </bb-cash-flow-forecast-settings-initial>
  </ng-template>
</ng-template>

This extension slot consumes the following object properties from its custom context:

Object property Description

hostRef

See API below

isError: boolean

Whether there is an error preventing the widget to load the forecast.

isLoading: boolean

Whether the forecast is loading.

forecastReferencesExists: boolean

Whether there is an active connection to an accounting provider. If there no active connection to an accounting provider, the widget shows a message that explains what is missing.

forecastData: ForecastItem[]
Untitled

The accounting data returned by the accounting provider. If there is an active connection to an accounting provider, but no accounting data, the widget shows a message that explains what is missing.

syncDatetime: string

Timestamp of the last data synchronization with the accounting provider.
