Contacts v3.4.3

This release note explains the changes to this capability.

New features

The new features in this release are:

  • Approval log in details view of my contact requests

    Bank customers now see the approval log for their own pending requests in the details view of the Manage contact change requests widget For each approver action the following details are displayed: approver name, date and time and approval status. For multi-approval policies the current user also sees if further approval is necessary.

  • Approval log in details view of pending approvals

    Bank customers now see the approval log for each contact approval request in Approve contact change requests widget For each approver action the following details are displayed: approver name, date and time and approval status. For multi-approval policies the current user also sees the details of previous approver(s).

FX Trading v3.4.3

New features

The new features in this release are:

  • The FX Rates Table widget FX Rates Table widget is now part of the Business Banking Widget Collection as the first FX Trading widget.

    This widget includes the following:

    • A table with FX quotes.

    • A pagination in desktop view and a load more button in mobile.

    • Buy and Sell buttons the bank customer uses to see which currency pairs are available for trading.

Bug fixes

The following issues have been fixed in this release:

  • CME-282 - The option to cancel a sweep in the Sweep Details widget was not accessible in the kebab menu using keyboard navigation.

  • CME-272 - The Sweep Details widget was not accessible using keyboard navigation.

Messages v3.4.3

This release note explains the changes to this capability.

New features

The new features in this release are:

  • Ability to attach files

    Bank customers can now attach files to messages in the Create Message and Message Thread widgets.

Notifications v3.4.3

This release note explains the changes to this capability.

New features

The new features in this release are:

  • The Notification Popup widget is now part of Business Banking Widget Collection.

    The Notification Popup widget provides customers with quick in-app updates on important matters. For instance, incoming payments to their accounts or changes in bank work hours.

Known issues

The known issues in this release are:

  • Introduced in v3.4.2

    • MAINT-7906 - An error is produced if the notification title is not set.

      Workaround: always set a title for notifications in one of the following places:

      • Notifications POST request payload

      • Action alert template

        If a meaningful title is not required, set a space symbol as the title.

    • MAINT-7907 - The title of each notification is not bold in Internet Explorer.

    • MAINT-7390 - The severity level icon is placed above the title.

  • Introduced in v3.4.3

    • DLRN-786 - When several new notification popups are displayed, the focus moves randomly between them.

    • MAINT-7905 - Notifications are not marked as read when the current user closes them using the close button in the popup.

Payments v3.4.3

This release note explains the changes to this capability.

New features

The new features in this release are:

  • Initiate Payment widget

    The Initiate Payment widget has been updated with bank branch code in order to handle Faster Payments and BACS payments. Bank customers use this feature to initiate single payments to UK accounts.

  • Initiate Business Payment widget

    The Initiate Business Payment widget has been updated with bank branch code in order to handle Faster Payments and BACS payments. Bank customers use this feature to initiate single payments to UK accounts.

  • Manage payments widget

    Manage payments widget allows users to view and manage ACH credit payments.

Products v3.4.3

This release note explains the changes to this capability.

Known issues

The known issues in this release are:

  • Introduced in v3.4.2

    • PRSUM-4927: Balance History widget BETA

      • The 'Export to CSV for the last 2 years' does not show the correct error message.

      • The x and y values are not correct when the current user filters the results.

    • PRSUM-5029: Balance History widget BETA

      • The legend in the graph displays 0.00 when the minimum value is negative.

Sweeping v3.4.3

This release note explains the changes to this capability.

New features

The new features in this release are:

  • Resizing of the sweep status badge

    The sweep status badge in the Sweep List widget now changes the size according to the screen resolution for better readability.

  • Menu icon is consistent with other capabilities

    The look of the menu icon for each individual sweep in the Sweep List widget is now consistent with other capabilities.

Bug fixes

The following issues have been fixed in this release:

  • CME-334 - When defaultStartingAtTime and defaultEndingAtTime properties were set to "12:00 PM", the corresponding "Starting at" and "Ending at" fields in the Create Sweep widget were empty.

Migration

There are no migration steps specific to this release.
