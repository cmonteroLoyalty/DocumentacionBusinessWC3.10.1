Configure forecast

Shared library

All Cash Flow Forecasting widgets use a shared library, see Cash Flow Forecasting common library.

Selector name

This widget has the following selector name: bb-cash-flow-configure-forecast-widget-ang

Required imports

The following imports are required to set up and customize this widget:

  • Widget module import

    Import the following modules into the module of your wrapper widget:

    import { CashFlowConfigureForecastWidgetAngModule } from '@backbase/business-ang/cash-flow';

  • Data module imports

    Import the following data modules into the module of your wrapper widget:

    import { ProductSummaryDataHttpModule } from '@backbase/data-ang/product-summary';
import { CashFlowDataHttpModule } from '@backbase/data-ang/cash-flow';

Extension slots

This widget has the following extension slots:

  1. bbCashFlowForecastConfigurationModalCustomizable

  2. bbCashFlowForecastConfigurationModalHeaderCustomizable

  3. bbConfigurationBankingAccountsPageCustomizable

  4. bbCashFlowForecastConfigurationModalFooterCustomizable

  5. bbCashFlowAccountingCategoriesPageCustomizable

  6. bbCashFlowAccountingCategoriesListCustomizable

  7. bbCashFlowConfigureForecastWidgetCustomizable (not in the screenshots)

Whether there are no accounts selected. The value false indicates that at least one account is selected.

Whether all accounts are selected.
